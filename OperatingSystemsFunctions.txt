One function of the operating system is to translate the computer's processes into a graphical interface. It makes using a computer easy to use because of how it organizes everything on the screen.
Another function is of an OS is to manage all ongoing processes that the CPU is handling. It allows the user to change and edit the tasks.
Another function of the OS is to run all programs and software. It also organizes everything so you can easily start up any application you want and have multiple running at once with a single click.
